# 基础用法
## 基础用法
### 基础用法
内容|功能
-|-
波纹|表示删除位置不留白
方向键|左右帧移动，上下前后视频断点处
分割|ctrl+b，选特定区域可以只剪特定区域
加速|对片段鼠标右键。
&emsp;|右边检查器可以裁剪区域等
&emsp;|三个基本转场，还有左上角的一个专门转场按钮
精确控制速度|右键，速度控制

### 剪辑界面

左边下角有时间线选项显示
